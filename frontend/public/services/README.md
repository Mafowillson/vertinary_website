# Service Images Directory

Place your service images in this directory with the following filenames:

1. **formation.jpg** - For "Formation en Ligne et Pratique" (Online and Practical Training)
2. **vente-animaux.jpg** - For "Vente d'Animaux" (Sale of Animals)
3. **accelerateur.jpg** - For "Accélérateur de Croissance Bio" (Bio Growth Accelerator)
4. **construction.jpg** - For "Construction des Bâtiments" (Construction of Breeding Buildings)
5. **projets-elevage.jpg** - For "Projets d'Élevage" (Livestock Projects)

## Image Requirements:
- Recommended format: JPG or PNG
- Recommended size: 800x600px or larger (will be automatically resized)
- Aspect ratio: 4:3 or 16:9 works best
- File size: Keep under 500KB for optimal loading

## Upload Instructions:
1. Save your images with the exact filenames listed above
2. Place them in this `/public/services/` directory
3. The images will automatically appear on the services page

## Alternative:
If you want to use different filenames, update the `imageUrl` paths in:
`/frontend/src/data/mockServices.js`
